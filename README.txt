	Keybind:
Escape -> clear the text input.
shift + Escape -> disconnect and exit.
Arrow Key -> move.
shift + Arrow Key -> append the direction to the text input.
Fn -> append the Nth macro to the text input.
shift + Fn -> save the text input as the Nth macro.
Return -> send the text input to the server.
Backspace -> remove the last character from the text input.
